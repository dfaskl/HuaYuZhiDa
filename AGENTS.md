# AGENTS.md

This file is the agent-facing working agreement for the `化语智答` repository.

## Purpose

化语智答（HuaYuZhiDa）是面向高校师生的智能问答系统，基于 RAG 架构构建：

- Spring Boot 后端
- Vue 3 + TypeScript 前端
- MySQL 主数据库
- Redis 缓存/会话/聊天上下文
- Elasticsearch 检索引擎
- Kafka 异步文件处理
- MinIO 对象存储

## Architecture

### Backend
- Java package: `com.huayu.smartqa`
- Entry point: `SmartQaApplication.java`
- REST API under `/api/v1/`
- WebSocket at `/chat/{token}`

### Frontend
- pnpm monorepo with `@sa/*` workspace packages
- Auto-imports via `unplugin-auto-import`
- Routes auto-generated by `@elegant-router/vue`

### Multi-tenancy
化语智答 uses department tags (院系) and user/dept relationships.
- `DeptTagAuthorizationFilter` enforces access control
- `DeptTagCacheService` manages dept tag hierarchy
- Resources carry `deptTag` for filtering

## Key Files

### Backend
- `config/SecurityConfig.java` - Security chain configuration
- `config/JwtAuthenticationFilter.java` - JWT validation
- `config/DeptTagAuthorizationFilter.java` - Dept-based access control
- `service/ChatHandler.java` - ReAct agent loop
- `service/HybridSearchService.java` - KNN + BM25 search
- `service/VectorizationService.java` - Embedding generation

### Frontend
- `store/modules/auth/index.ts` - Authentication state
- `store/modules/chat/index.ts` - WebSocket chat
- `service/request/index.ts` - API client configuration
- `router/guard/route.ts` - Route guards

## Development

### Commands
```bash
# Backend
mvn spring-boot:run
mvn test

# Frontend
cd frontend
pnpm dev
pnpm typecheck
pnpm lint
```

### Configuration
- Backend: `.env` file in project root
- Frontend: `frontend/.env`, `frontend/.env.test`, `frontend/.env.prod`

## Conventions

### Code Style
- Backend: Lombok for boilerplate reduction
- Frontend: Composition API with `<script setup>`
- All API responses follow `{ code, message, data }` pattern

### Naming
- Java classes: PascalCase
- Java methods/variables: camelCase
- Frontend components: kebab-case
- API paths: kebab-case
- Database tables: snake_case
- ES fields: camelCase

### Git
- Pre-commit hooks run typecheck and lint
- Conventional commits in Chinese
